1. Install all the tools that I gave in the Tools Used Folder
2. While Installing The MySQL server set the password as root
3. 1 DataBase is created with name as game;
4. The database contains 3 tables.
i) Players Table: Stores Information of players registered.
ii) GameResults Table: Store all the winners and losers of the games with timestamp
iii) ErrorLog Table: To store all the Runtime Exceptions, so that they can be removed in the next version of the project and give better experience to user.
5. Make sure that Java and MySql are connected by JDBC.
6. Go To the Main class and execute it from the IntelliJ IDE to be able to access the application.
7. Please run mySql Server in localHost, If Db is not created by itself run code create database game;
